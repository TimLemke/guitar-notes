Learn Yer Guitar Notes!

This is an exercise to help you learn where the notes are on the guitar fretboard: it will show you a random note to play on a random string. It assumes a 6-string guitar in standard tuning, EADGBe, and the higher E string will be written lowercase.

Select which note collections you want to include (at least one). You can either get single notes at a time, or start a timer, which will show you a new note every 5 seconds.
